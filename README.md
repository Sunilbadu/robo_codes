# 🤖 Robotics IDE Codes

Welcome to the **Robotics IDE Codes** repository! This repository contains all of my robotics-related source code, experiments, and projects developed using various microcontrollers and development boards.

## 📌 About

This repository serves as a centralized collection of code for different robotics projects, including:

- 🚗 Line Following Robot
- 🧱 Wall Following Robot
- 🚧 Obstacle Avoiding Robot
- 📏 Ultrasonic Sensor Projects
- 🔥 IR Sensor Projects
- ⚙️ Motor Driver (L298N, TB6612FNG, etc.)
- 📡 ESP32 Wi-Fi Projects
- 🔌 Arduino UNO/Nano Projects
- 🤖 Autonomous Robot Algorithms
- 🛠️ Sensor Testing Codes
- 🧪 Experimental Robotics Programs

As I build more robots and test new ideas, this repository will continue to grow.

## 📥 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/<your-repository>.git
```

2. Open the desired project in your IDE.

3. Install the required libraries.

4. Select the correct board and COM port.

5. Upload the code to your microcontroller.
